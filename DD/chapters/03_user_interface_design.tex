\chapter{User Interface Design}

This chapter presents visual representations of the mobile application's user interface, covering all main user flows. Each mockup corresponds to the UI specifications previously described in the RASD document for the BBP platform and carefully reflects the sequence diagrams illustrated in the Runtime View section (see Section~\ref{sec:runtime-view}). The sections are organized by flow, with screens arranged from left to right and continuing across rows. User experience was a central consideration throughout the design of these interfaces and interactions.
